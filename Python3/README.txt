In this program you can get information about past earthquakes. You must enter all the necessary parameters in the format:
1) Date: YYYY-MM-DD
2) Longitude: 0.00 or -0.00
3) Latitude: 0.00 or -0.00
4) Maximum radius km: 0
5) min magnitude: 0


P.S.
0 - arbitrary digit
